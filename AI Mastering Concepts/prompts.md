# Prompts Used

## Prompt 1
Explain pointers in C using a simple analogy. I want to understand how a function can change two integers from the main by receiving their addresses instead of their values. Do not give the final code yet; focus on the concept.

## Prompt 2
For a function whose goal is to swap two integers using pointers, what is the correct step-by-step logic and syntax idea? Please explain what the dereference operator * does at each step, but do not write the complete final program.

## Prompt 3
Act like a security auditor. What is one important security or reliability risk in a pointer-based swap function, and how should I think about preventing it? Please focus on NULL checks or invalid memory access rather than the final code.

## Prompt 4
What edge case or C convention should I think about for the return value of a pointer-based swap function? For example, what should the function do or return if one input pointer is NULL?
