# AI: Mastering Concepts

## Task Description
This task focused on using AI as a conceptual tutor and security auditor to understand pointers in C. The goal was to learn how a function can swap two integers by receiving their addresses, while also thinking about NULL pointers, invalid memory access, and safe return-value behavior.

## Concept Chosen
Pointers

## Programming Goal
Understand how to design a function that swaps two integers using pointers, while carefully considering dereferencing, NULL checks, invalid memory access, and safe return-value conventions.

## AI Tool Used
ChatGPT

## Prompt Sequence Summary
1. Concept explainer: pointers using a simple analogy
2. Logic and syntax idea for swapping two integers using pointers
3. Security audit focusing on NULL checks and invalid memory access
4. Edge case and return-value convention for a pointer-based swap function

## Outcome
This task improved my understanding of pointers as memory addresses rather than just syntax. It also showed how AI can help break down a difficult concept into smaller learning steps while keeping safety and reliability in focus.
