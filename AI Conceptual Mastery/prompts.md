# Prompts Used

## Prompt 1
Compare Arrays and Linked Lists in C using a table format.

Your comparison must include at least the following features:
- Access Time
- Insertion/Deletion Efficiency
- Memory Usage
- Flexibility (size changes)
- Cache Performance
- Best Use Case

## Prompt 2
Explain the underlying memory mechanics of Arrays vs Linked Lists in C.

Specifically explain:
- Why arrays require contiguous memory
- How linked lists store elements using pointers
- The overhead of pointers in linked lists
- How memory allocation affects performance in both

Focus on deep understanding, not just definitions.

## Prompt 3
Provide a Big O comparison between Arrays and Linked Lists in table format.

Include both Time Complexity and Space Complexity for the following operations:
- Access (read element)
- Search
- Insertion
- Deletion

Your table must clearly compare:
- Array (Time & Space)
- Linked List (Time & Space)
